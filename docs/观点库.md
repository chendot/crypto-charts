---

#### 论点演进（入库前的推理链）

旧叙事（被推翻）：
  稳定币流入DeFi = 资金观望 = 看涨信号

数据事实：
  Lending_pct（Aave+Compound稳定币TVL/全链总量）
  - 2021-03牛市中段：12.3%
  - 2022-12熊市底部：1.5%
  - 2025-09牛市顶部：15.6%（历史新高）
  - 2026-04当前：4.8%

机制链条：
  借款需求旺盛（市场想加杠杆买BTC）
    → 以ETH/BTC为抵押借出稳定币
    → 借款利率上升
    → 吸引更多稳定币存入Lending
    → Lending TVL上升
  ∴ Lending TVL高 = 杠杆需求旺盛的代理变量
    不是"子弹没用完"，而是"子弹已经上膛"

反共识：
  旧解读：链上稳定币多 = 还有子弹 = 看涨
  新解读：Lending TVL创新高 = 杠杆已满 = 距离顶部不远
  2025-09 Lending_pct 15.6%创历史新高，当时市场普遍兴奋——这就是认知幻觉
