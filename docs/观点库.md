- 系列：破例（宏观信号 × DeFi 资金面）
- 状态：💭 待发酵
- 优先级：中
- 数据源：DeFiLlama /protocols (stablecoinTvl by category), BTC 价格 (CoinGecko)
- 数据可获取性：🔍 需采集（需按协议分类做时间序列，当前快照已验证）
- 实战背书：⭐⭐ 旁观（数据驱动，无直接持仓经验）
- 一句话观点：稳定币不是"不小心留在DeFi"——RWA和基差交易提供了链上替代收益，稳定币持有者主动选择了比买BTC更确定的出口，分流了原本会流向spot的资金
- 快照数据（2026-05-12）：
  - 全链稳定币总量 $321.21b
  - DeFi协议稳定币TVL $61.58b，占比 19.2%
  - 分类结构：RWA $12.91b + Bridge $12.70b + Lending $7.46b + Basis Trading $6.70b + Risk Curators $4.71b + CDP $4.15b + DEX $4.00b
  - RWA + Basis Trading 合计 $19.6b，占DeFi锁仓32%——这部分有明确的"不买BTC"意图
- 核心因果链：
  - 美联储加息(2022) → 链上美债收益出现(2023) → RWA/基差策略TVL爆发
  - → 稳定币有了BTC之外的收益出口
  - → 单位稳定币对BTC的买入转化率下降
- 待验证：
  - RWA + Basis Trading 分类TVL时间序列——是否在2023年后系统性爬升？
  - 2021牛市顶部的DeFi锁仓占比基线——19.2%是高还是低？
  - 历史数据脚本待写，验证完再决定是否升格D-04
- 备注：
  - 论点已从"相关性减弱"升级为"有因果链的结构性分流"——更硬
  - 原"DeFi内循环"叙事过于笼统，现在的叙事更窄也更准确
  - 不要用"传导效率下降"作为正文术语——图表内用"每新增$1稳定币流入BTC的比例"更直觉
